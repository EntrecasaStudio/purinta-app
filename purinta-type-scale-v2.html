Purinta Type System v2

Ohno Softie + Rubik

Type system based on Major Third scale (1.25) with 16px base. Ohno Softie as display font (titles, amounts, buttons) and Rubik as UI font (body, labels, data). Optimized for the Purinta mobile-first DApp.

00 — Foundations
1.25 Scale — Major Third
Each step is 1.25x the previous. 13 levels from 3xs (8px) to 7xl (119px). Line-height adjusts inversely: large titles 1.2x, body 1.4x, small text 1.4x.
1.250
Major Third

Balanced harmonic ratio for UI. Generates clear hierarchical contrast without extreme jumps. Ideal for financial interfaces where readability is critical.

16px
Base Size (sm)

Base size for body and buttons. WCAG AA compliant. Step 0 of the scale. All other sizes derive from this.

0.5px
Letter Spacing

Ohno Softie requires 0.5px global letter-spacing due to its compact x-height. Buttons use 0.8px. Uppercase tags use 1.0px.

01 — Display Scale
Ohno Softie — Titles, Amounts, Buttons
Kawaii display font with rounded terminals. Weight 400 by default (Ohno Softie already has visual presence). Hero uses 600 for emphasis. Letter-spacing 0.5px global, 0.8px on buttons. Minimum recommended size: 16px (sm). Below 16px, Ohno Softie loses legibility due to its rounded terminals and compact x-height — use Rubik instead for smaller sizes.
